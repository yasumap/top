# AGENTS.md

このファイルは、やすまっぷ（yasumap）のLPに関する意図・要件のメモです。必要に応じて追記・修正してください。

## プロジェクト意図
- Next.js（App Router）+ TypeScript + Tailwind CSS で静的LPを作る。
- LPは機能説明ではなく、思想やストーリーを静かに伝える場。
- 売り込み感や自己主張を避け、やさしく落ち着いた空気感を優先。
- 余白・雰囲気・読みやすさを最優先。アニメーションは最小限。

## デザイン方針
- 背景は提供画像（`public/夕日のベンチ_LP用背景画像.png`）をそのまま見せる。
- 網掛けや強いオーバーレイは使わない。必要ならごく薄い暗さのみ。
- 文字は白基調で読みやすく。背景パネルは使わない方針。
- 1カラム構成。スクロールに従ってテキストが順番に現れる。
- フェードインはゆっくりだが、俊敏さを上げたスピード感で。
- スクロールで上に戻したときはフェードアウトする挙動。

## コピー/文言の方針
- 「夕暮れ」「夕焼け」など時間帯に限定される語は使わない。
- 押しつけがましい表現、過剰な感情の煽りは避ける。
- ヒーロー文言は2行で表示する（例：「街角に、やさしい」「ベンチの地図を。」）。
- ストーリー全文は指定文を使う。1〜2行単位に分割してフェードイン。

## セクション構成
1. ヒーロー（背景画像 + 短いキャッチ + 説明文）
2. スクロールセクション（短文を段階的に表示）
3. フッター（やすまっぷ + 「現在開発中・準備中」）

## 実装メモ
- 画像: `public/夕日のベンチ_LP用背景画像.png`
- ファビコン: `public/椅子_ファビコン用.png` を `app/icon.png` と `app/apple-icon.png` に配置
- フェード: IntersectionObserverでisIntersectingに応じて表示/非表示
- カード背景は薄めの透かしに調整
